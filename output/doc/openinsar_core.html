openinsar_core.xml_utils module¶

Utilities for working with XML files.

OpenInSAR uses XML files for a variety of purposes. As such this bespoke XML interface is useful. In general, however, XmlObject is basically a wrapper around the ElementTree library.

class openinsar_core.xml_utils.XmlObject(file: str | None = None, xml_string: str | None = None, xml: ~xml.etree.ElementTree.Element = <Element 'root'>)¶

Bases: object

A class that represents an XML object and provides methods to manipulate it.

While this mainly uses the ElementTree library, this class allows us to adapt much more efficiently if something should change in that library.
